# Mutx163 个人主页

这是我的个人主页网站，展示了我的个人信息、技能和项目经历。

## 网站特点

- **响应式设计**：适配各种屏幕尺寸，包括手机、平板和桌面电脑
- **现代UI**：使用Tailwind CSS构建的现代美观界面
- **交互体验**：包含平滑滚动、悬停效果和动画
- **技能图表**：使用Chart.js可视化展示技能水平
- **联系表单**：提供方便的留言功能

## 如何运行

### 方法一：直接打开HTML文件

直接双击 `index.html` 文件即可在浏览器中打开。

### 方法二：使用本地服务器

推荐使用Python的简易HTTP服务器来运行：

```bash
python -m http.server 8000
```

然后在浏览器中访问 `http://localhost:8000`
